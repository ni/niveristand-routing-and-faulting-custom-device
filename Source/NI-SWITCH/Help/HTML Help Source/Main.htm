SLSC Switch

The SLSC Switch Custom Device is an add-on that supports SLSC routing modules.

The SLSC Switch Custom Device enumerates all available endpoints for a routing module. Set the configuration for each endpoint to inform the SLSC Switch driver of valid endpoints to connect to.

The following table describes the available endpoint configurations.

Configuration Description
Load A load endpoint is unrestricted and can be connected to other load and source endpoints. This is the default configuration for an endpoint.
Source A source endpoint is restricted and cannot be connected to another source endpoint.
Configuration A configuration endpoint is reserved by the hardware for internal routing. This configuration cannot be used directly.

NOTE: Configure the SLSC Switch Custom Device before configuring the Routing and Faulting Custom Device.

What do you want to do?

Import SLSC Switch Capabilities

Click Autodetect to import switching capabilities from the SLSC Switch module. The SLSC Switch module must online to automatically detect the capabilities. Click Load from JSON to import a switching capabilities file from disk. Import the capabilities from from file if the SLSC Switch module is not present.

Script SLSC Switch Configuration

Open the scripting examples and run Configure Modules from Hardware.vi to configure one or more SLSC Switch routing modules in the system definition file.
