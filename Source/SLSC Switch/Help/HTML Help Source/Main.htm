SLSC Switch

The SLSC Switch Custom Device supports SLSC Switch routing modules by enumerating all available endpoints, and making them available to the Routing and Faulting Custom Device.

Set endpoint configurations to modify the possible connections for the routing module. The following table lists the available endpoint configurations.

Configuration Description
Load Endpoint is unrestricted and can be connected to other load and source endpoints. This is the default configuration.
Source Endpoint is restricted and cannot be connected to another source endpoint.
Configuration Endpoint is reserved for internal hardware routing. This configuration cannot be used directly.

NOTE: Configure the SLSC Switch Custom Device before configuring the Routing and Faulting Custom Device.

What do you want to do?

Import SLSC Switch Capabilities

Click Autodetect to import switching capabilities from the SLSC Switch module. The SLSC Switch module must online to automatically detect the capabilities. Click Load from JSON to import a switching capabilities file from disk. Import the capabilities from from file if the SLSC Switch module is not present.

Script SLSC Switch Configuration

Open the scripting examples and run Configure Modules from Hardware.vi to configure one or more SLSC Switch routing modules in the system definition file.
