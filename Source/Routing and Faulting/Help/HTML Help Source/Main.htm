Routing and Faulting

The Routing and Faulting Custom Device is an add-on that configures endpoint aliases and routing channels. It controls multiple pieces of routing hardware, such as certain SLSC modules, in a coordinated manner.

What do you want to do?

View/Edit Routing and Faulting Configuration

Configure routing channels by adding them to the Routing Channels section. Use the Endpoint Aliases section to provide convenient names for hardware resources.

Script Routing and Faulting Configuration

Open the scripting examples and run Configure Routing and Faulting Custom Device.vi to add routing channels, states, and connections.
