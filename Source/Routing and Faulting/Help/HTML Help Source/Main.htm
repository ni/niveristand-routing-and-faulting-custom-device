Routing and Faulting

The Routing and Faulting Custom Device is a VeriStand add-on that configures aliases and routing channels.

What do you want to do?

Script Routing and Faulting Configuration

Open the scripting examples on disk from Source/Scripting Examples/Scripting Examples.lvproj. Run Configure Routing and Faulting Custom Device.vi to add routing channels, states, and connections.
